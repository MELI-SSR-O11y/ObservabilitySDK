⚠️ **ADVERTENCIA IMPORTANTE** ⚠️

Los artefactos `.aar` de este directorio están precompilados y listos para una prueba rápida.

**Punto Clave:** Estos artefactos están configurados para apuntar a la dirección IP estática `192.168.1.3`.

**¿Qué significa esto para ti?**
*   Si tu servidor de backend no se encuentra en esa dirección IP, las peticiones de red fallarán.
*   Para conectar la librería con el backend de este proyecto (o con tu propio servidor local), es **esencial** que compiles un nuevo artefacto.

**Acción Requerida:**
Por favor, sigue las instrucciones de compilación detalladas en el archivo `README.md` para generar un nuevo `.aar` con la configuración de red correcta para tu entorno.
